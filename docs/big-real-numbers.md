## **Big Real Numbers**

This is not an **extended fixed precision floating point** type
but rather a **dynamic precision floating point**.

### **Headers**

- **Header Mode :** `include "APA/core.hpp"`

- **Static Lib :** `include <APA/bfloat.hpp>`

-----

<br>

### **Namespace**

- **`apa::bfloat`**
